# DDPM on MNIST — Conditional Denoising Diffusion Model

This repository provides an implementation of a **Denoising Diffusion Probabilistic Model (DDPM)** trained on the **MNIST** handwritten digits dataset.  
The model supports **class-conditional image generation** using classifier-free guidance.



## Features

- TensorFlow/Keras implementation of DDPM  
- Class-conditional sampling (digits 0–9)  
- Uses classifier-free guidance  
- Trains on MNIST (28×28 grayscale)  
- Includes sampling utilities and visualization code  
- Clean notebook implementation for experimentation
